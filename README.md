# tmarket

![Sample Screenshot](https://github.com/echohtp/tmarket/blob/main/image.jpg?raw=true)


### Prerequisites

- Node v18.18.0 or higher

### Installation

#### Clone the repo

```shell
git clone <repo-url>
cd <repo-name>
```

#### Install Dependencies

```shell
npm run install
```

#### Start the web app

```
npm run dev
```
